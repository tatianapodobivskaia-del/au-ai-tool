# Socratic Tutor

A thinking partner for university students.

## About

Socratic Tutor is an academic research prototype built as part of the Git & GitHub Workshop at Atlantis University. 

Instead of giving direct answers, this tool teaches by asking guiding questions — helping students think through problems on their own. Inspired by the Socratic method of teaching through inquiry.

## How It Works

Students ask a question or describe a concept they want to understand. Instead of providing the answer, the tutor responds with thoughtful questions that guide the student to discover the answer themselves. This approach builds critical thinking skills and deeper understanding.
